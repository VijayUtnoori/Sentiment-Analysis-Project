How to Use Sentiment Analysis System

1) Use the Dashboard
  • The dashboard allows you to analyze text or upload files for sentiment analysis.
  • You can also view results, trends, and history from the dashboard.
2) Text Sentiment Analysis
  • Enter your text in the input box.
  • Click the Analyze button.
  • The system will show Positive, Negative, or Neutral sentiment.
3) File Upload Analysis
  • Click the Upload File button.
  • Select CSV or TXT file only.
  • Maximum allowed file size is 5MB.
  • Click Analyze to process the file.
4) Results and Graphs
  • The system shows total sentiment count.
  • Shows Positive, Negative, and Neutral counts.
  • Graphs help visualize the results.
5) History
  • You can view your previous analysis in History section.
  • This helps track your sentiment analysis activity.
